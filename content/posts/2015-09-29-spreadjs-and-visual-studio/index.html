You can use SpreadJS in a Visual Studio project. This provides access to automatic complete and code checking in Visual Studio.

Use the following steps to create a project that uses SpreadJS.

  1. Create an ASP.NET project (File, New Project or New Project in Visual Studio).ï¿½ Select ASP.NET Web Application. Specify the Name, Location, and Solution name.

    • Select a template such as Empty and select OK.

      • Select Project, Add New Item.

        • Select HTML page. Specify a name. Select Add. You can also right-click on the HTML file in the Solution Explorer and change the name (Rename menu option).

          • Save the file (File, Save SpreadJSHtml.html).

          • Add the file to the project (Project, Add Existing Item).

            • Select the file in the project folder (SpreadJSHtml.html) and select Add. You can also use the Add Existing Item option to add the gcspread.sheets.all.8.40.20151.5.min.js and gcspread.sheets.all.8.40.20151.5.css files to the project so that you can use relative paths.
